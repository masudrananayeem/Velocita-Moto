## 🚀 Core Features

| # | Feature | Details |
|---|---|---|
| 1 | **Hero Carousel** | 10 luxury car slides with specs (HP, Speed, 0–60); custom indicators with pulse animation; auto-play with hover pause |
| 2 | **Brand Directory** | 12+ brand cards with hover effects; dropdown with 26+ categorized brands; direct links to official websites |
| 3 | **Car Collection** | Filter by All, Hypercar, Supercar, Luxury, Performance, American, Japanese; Load More functionality; country of origin display |
| 4 | **Interactive Counters** | Animated stats on scroll across 4 metrics: Models, Brands, Luxury, Reviews |
| 5 | **Video Integration** | Full-width banner video; background video section with overlay |
| 6 | **Luxury Showcase** | 12 premium car cards with hover zoom and overlay effects |
| 7 | **Testimonials** | 3 testimonials with avatars and star ratings |
| 8 | **Quote Section** | Sir Henry Royce quote with parallax background |
| 9 | **Contact** | Contact form, Google Maps integration, contact details |
| 10 | **Navigation** | Fixed navbar with sidebar, animated menu links, smooth scroll |
| 11 | **Footer** | Quick links, social icons, developer signature |

## 🔧 Installation

### Prerequisites

- Any modern browser
- VS Code (recommended)

### Steps

**1. Clone or Download**
```bash
git clone [repository-url]
cd velocita-moto
```

**2. Run with Live Server**
```
Right-click index.html → Open with Live Server
```

**3. Customize**
- Replace images in `img/` folders
- Update brand links in dropdown
- Edit content in HTML files
